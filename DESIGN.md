# Quizlet — Style Reference
> Academic Playground on Soft Gray. Like a well-organized desk scattered with colorful learning tools.

**Theme:** light

This design system feels like a friendly, structured learning environment, prioritizing clarity and interactive engagement. The dominant near-gray background `#f6f7fb` provides a clean canvas, while a bold violet `#4255ff` defines primary interactive elements. The app should feel educational and approachable without becoming decorative.

## Typography

- Quizlet uses `hurme_no2-webfont`; this app uses Pretendard for Korean readability.
- Substitute stack: `Pretendard`, `system-ui`, `sans-serif`.
- Use 700 for page titles and major action text.
- Keep letter spacing normal.

## Shapes And Spacing

- Base unit: `8px`
- Cards: `8px` radius
- Inputs: `4px` radius
- Buttons: `200px` pill radius
- Main card shadow: `rgba(40, 46, 62, 0.1) 0px 4px 16px 0px`
- Major vertical spacing: `48px`
- Card internal padding: `16px`

## App Adaptation

This is a school work app, so avoid excessive decorative illustration. Use the Quizlet feel through:

- soft gray background
- clean white cards
- violet primary actions
- compact pill buttons
- readable status badges
- clear mobile-first cards
- simple admin data layout
